# Which micro-controller should I use for my project?

## First Time programming a microcontroller:

    Arduino Uno, Arduino Nano, Feather MO

----
## Audio

    Teensy, Rasberry Pi, Bela, ESP32 (with audio module)

    Play MP3/WAV files: WAV Trigger, Tsunami WAV Trigger, Arduino Uno with MP3 shield
    
----
## Bluetooth

    ESP32 (built-in Wi-Fi & Bluetooth)
----
## Computer Vision

    Raspberry Pi
----
## Display (OLED, TFT, LCD)

  Small displays: Arduino boards, ESP32
  
  Large / HD display: ESP32, Raspberry Pi, RPi Pico (RP2040), Teensy

----
## LED strips Addressable (WS2812 / NeoPixel)

    If short LED strip: Arduino boards

    If long LED strip (hundred+): ESP32, RPi Pico (RP2040), Teensy

    Multiple LED strips: Feather Scorpio
    
----
## SD Card Module
  
    ESP32, Raspberry Pi, RPi Pico (RP2040), Teensy

    Avoid Arduino UNO (not enough RAM)

----
## Networking (Ethernet / WIFI)

    Arduino (with Ethernet/WIFI shield), Arduino Wifi, ESP32, RPi

----
## Motors

    Single control: Arduino Boards, ESP32

    Multiple motors: ESP32, Teensy
----
## Sensors

    Simple control: Arduino Boards, ESP32

    Multiple sensors: ESP32, Rpi, Rpi Pico, Teensy

----
## Touch - Capacitive boards

    Bare Conductive Touch Board

    with Adafruit 12-Key Capacitive Touch Sensor
----
## Wearables 

    Arduino Lilypad, Arduino Gemma / Flora, Arduino Nano 33 BLE, ESP32 (specific boards for wearables)
